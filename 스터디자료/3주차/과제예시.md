# 과제 예시- 4팀 최규민 3주차 과제

## TinkerCAD 회로 및 코드
- 스위치 실습\
프로젝트 링크 : https://www.tinkercad.com/things/kfLMqFfdtYg

```c
int led = 13;
int sw = 4;

void setup() {
  pinMode(led, OUTPUT);
  pinMode(sw, INPUT);
}

void loop() {
  if (digitalRead(sw) == HIGH) {
    digitalWrite(led, HIGH);
  }
  else {
    digitalWrite(led, LOW);
  }
}
```
